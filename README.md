# 📊 Sales & Profitability Dashboard (Power BI)

## Overview
This project is an interactive Power BI dashboard developed using a retail sales dataset containing 1,000 records. The dashboard provides actionable business insights through dynamic KPIs, interactive filters, and multiple visualizations.

## Key Features
- 📈 Sales & Profit Trend Analysis
- 💰 KPI Cards (Sales, Profit, Orders, Profit Margin, Average Order Value)
- 🌍 Regional & State-wise Sales Analysis
- 🛍️ Category & Sub-Category Performance
- 🎯 Interactive Slicers (Date, Region, Category, Segment, Payment Mode)
- 📊 Advanced DAX Measures & Calculated Columns
- 📍 Executive Dashboard Design

## Tools Used
- Microsoft Power BI
- DAX (Data Analysis Expressions)
- Power Query
- Data Modeling
- Excel/CSV Dataset

## Skills Demonstrated
- Data Cleaning
- Data Modeling

- Dashboard Design
